# 132-33-KV-SUBSTATION


132/33 kV Substation Single Line Diagram (SLD)

Overview

This repository contains a 132/33 kV Substation Single Line Diagram (SLD) designed for educational, academic, and reference purposes. The diagram illustrates the arrangement of major substation equipment, bus configurations, transformers, feeders, protection devices, and capacitor banks used in a typical 132/33 kV power substation.

Features

- 132 kV Main Bus and Transfer Bus arrangement
- Dual 132 kV Incoming Transmission Lines
- Two 63 MVA Power Transformers
- 33 kV Main Bus System
- Eight 33 kV Outgoing Feeders
- 10 MVAR Capacitor Banks
- Current Transformers (CT)
- Capacitive Voltage Transformers (CVT)
- Lightning Arresters (LA)
- Circuit Breakers (CB)
- Isolators (ISO)
- Station Transformer Connection
- Color-coded voltage levels:
  - Red – 132 kV System
  - Green – 33 kV System
